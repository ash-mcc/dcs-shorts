Asciidoc -> HTML
	asciidoctor -b html5 README.adoc
	...out-of-the-box, asciidoctor generates nicer HTML5 than asciidoc 

HTML -> PDF
	open in a web browser, the HTML file that was generated by the previous step 
          and then ask the web browser to export the HTML page as PDF 

Asciidoc -> PDF
	asciidoctor -r asciidoctor-pdf -b pdf -a allow-uri-read README.adoc
